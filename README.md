# 📸 PhotoBooth Web App ✨

Ứng dụng chụp và trang trí ảnh lấy liền ngay trên trình duyệt với giao diện **Galaxy Glassmorphism** cực chill! 🌌

## ✨ Tính năng chính
- **Chụp ảnh sắc nét**: Hỗ trợ camera tỉ lệ 4:3 độ phân giải cao, tự động đếm ngược.
- **Bộ lọc màu (Filters)**: Chỉnh màu live với các bộ lọc như Hoài cổ, Hạt Film, Hồng đào, v.v.
- **Khung & Sticker**: Tự động crop ảnh vừa vặn vào khung (Cute/Film/Fresh) và thoải mái trang trí, xoay, thu phóng sticker.
- **Lưu ảnh High-DPI**: Xuất file ảnh nét căng để sống ảo.

## 🚀 Hướng dẫn sử dụng
Do chính sách bảo mật Camera của trình duyệt, web **bắt buộc** phải chạy trên Server (localhost hoặc HTTPS).
1. Mở thư mục code bằng **Visual Studio Code**.
2. Cài đặt extension **Live Server**.
3. Chuột phải vào file `index.html` -> Chọn **"Open with Live Server"**.
4. Cấp quyền truy cập Camera trên trình duyệt và bắt đầu chụp!

*(Hoặc đơn giản nhất là host trực tiếp lên GitHub Pages để có link xài luôn).*
